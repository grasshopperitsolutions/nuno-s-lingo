## Brief overview
This rule file defines mandatory internationalization and SEO requirements for all content additions and modifications in this project. These rules apply to every development task involving user-facing content.

## Content implementation requirements
  - Always check for existing translations before adding new text content
  - Verify appropriate language handling is in place for all new content
  - Include proper SEO considerations specific to the target language(s) when adding or modifying page content
  - Never hardcode raw text directly in components - always use translation system patterns
